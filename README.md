# 🧠 Jeu de Memory - Projet BTS SIO SLAM

## 📌 Description

Ce projet est une application de bureau développée en **C# avec WinForms** dans le cadre du BTS SIO option SLAM.

Il s'agit d'un jeu de Memory (jeu de paires) configurable permettant au joueur de personnaliser son expérience de jeu en choisissant :

* Le nombre de paires de cartes
* Le thème des images

Le but du jeu est de retrouver toutes les paires en un minimum de coups et de temps.

---

## ⚙️ Fonctionnalités

### 🎮 Configuration de la partie

* Choix du nombre de paires de cartes
* Sélection d’un thème d’images

### 🃏 Gameplay

* Disposition aléatoire des cartes sur une grille
* Retourner deux cartes à la fois
* Vérification automatique des paires

### 📊 Suivi de la partie

* Compteur de coups
* Chronomètre en temps réel

### 🔄 Dynamique de jeu

* Cartes mélangées à chaque nouvelle partie
* Interface interactive et responsive
  
---

## 🎮 Règles du jeu

* Retourner deux cartes à chaque tour
* Si les cartes sont identiques → la paire est validée
* Sinon → elles sont retournées face cachée
* Le jeu se termine lorsque toutes les paires sont trouvées
